    dataview-stream

    Description

    Installation

    npm install dataview-stream

    API

    Full API documentation is available.

    Supported read methods:

    • bytes(len)
    • ascii(len)
    • utf8(len)
    • u8()
    • u16()
    • u32()
    • u64()
    • i8()
    • i16()
    • i32()
    • i64()
    • f16()
    • f32()
    • f64()

    Each of those reads the appropriate number of bytes, then advances the read position by that number of bytes. An exception will be thrown if you go past the end of the input.

    Example:

    import {DataViewStream} from 'dataview-stream';

    const buf = new Uint8Array([1, 2, 3, 4]);
    const dvs = new DataViewStream(buf);

    dvs.u8(); // 0x01
    dvs.u16(); // 0x0203
    dvs.reset(); // Go back to the beginning
    dvs.u32(); // 0x01020304
